# Azure-RAG-Ingestion-API
A document ingestion service built with Spring Boot and Spring Cloud Azure. It automates the process of secure PDF uploading to Azure Blob Storage and programmatically triggers Azure AI Search indexers to ensure near real-time data availability for downstream AI applications.

See [Azure RAG App](https://github.com/AndrewBenson98/Azure-RAG-App) for more details on this project
